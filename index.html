Веб-приложение для прогноза клёва леща в Днепре (Киев) и погоды

Проект создан с нуля как специализированный сервис для рыбаков, объединяющий метеоданные и астрономические расчёты. Основная цель – предоставить удобный и наглядный прогноз клёва, учитывающий погодные условия и фазу Луны.

перейти на сайт

Функциональность

Приложение формирует прогноз клёва на основе погодных факторов и биоритмов рыбы:

  • Прогноз на утро, день и вечер.
  • Долгосрочный прогноз на 5 дней вперёд.
  • Детализация погоды: температура, ветер, давление, осадки, влажность.
  • Расчёт фазы Луны и дня лунного цикла с помощью астрономической формулы.
  • Архив погоды для анализа прошлых условий.

Фронтенд и алгоритмы

Проект разработан на Vanilla JavaScript с использованием API Open-Meteo. Основные особенности:

  • Алгоритмы анализа давления, температуры, ветра и осадков + нормализация осадков по архиву.
  • Учёт лунной активности и сезонности для каждого периода дня.
  • Формирование итоговой оценки клёва: отличный, хороший, слабый или отсутствует.
  • Минималистичный интерфейс и чёткая структура разделов: «Прогноз клёва», «Прогноз погоды», «Архив».

Собственные JSON-данные (доменные правила)

Помимо данных из API, приложение использует набор локальных JSON-файлов, которые дополняют расчёты и позволяют учитывать реальные поведенческие паттерны леща:

  • calendar.json — календарь сезона (весна/лето/осень) и связанные коэффициенты.
  • calendar-moon.json — лунный календарь с днями цикла.
  • terms-moon.json — словарь лунных терминов (например, T_NEW_MOON, T_YOUNG_MOON) с описаниями влияния на клёв и поминутной детализацией для утро/день/вечер.
  • bite-rules.json — экспертные правила по периодам с рекомендациями по снастям и тактике:

    Эти файлы подключаются на этапе инициализации и участвуют в агрегировании факторов (погода → сезон → луна → поведенческие рекомендации), повышая точность итогового прогноза и его практическую ценность.

    Технологии

    • JavaScript для логики и обработки данных.
    • Open-Meteo API для прогноза и архива погоды.
    • Астрономический алгоритм для вычисления фаз Луны.
    • Gulp + Webpack для сборки проекта.
    • GitHub Pages для хостинга.

    Результаты

    Получено удобное веб-приложение для планирования рыбалки на Днепре в Киеве. Прогноз опирается на комбинацию метеофакторов, лунного цикла и локальных экспертных правил в JSON, что делает выдачу более реалистичной и прикладной.

    Мой вклад

    Разработал архитектуру и алгоритмы, реализовал фронтенд на чистом JS, настроил сборку (Gulp/Webpack) и развернул на GitHub Pages. Подготовил и интегрировал собственные JSON-словарь/правила.
